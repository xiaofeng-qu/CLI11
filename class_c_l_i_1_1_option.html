◆ get_name()

std::string CLI::Option::get_name ( bool  positional = false,
bool  all_options = false 
) const
inline

Gets a comma seperated list of names. Will include / prefer the positional name if positional is true. If all_options is false, pick just the most descriptive name to show. Use get_name(true) to get the positional name (replaces get_pname)

The all list wil never include a positional unless asked or that's the only name.

◆ type_size_

int CLI::Option::type_size_ {1}
protected

The number of arguments that make up one option. -1=unlimited (vector-like), 0=flag, 1=normal option, 2=complex/pair, etc. Set only when the option is created; this is intrinsic to the type. Eventually, -2 may mean vector of pairs.
